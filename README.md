# 💸 Personal Expense Tracker

Ever looked at your bank account at the end of the month and wondered where all your money went? Same here! 😅 

This is a simple, beautiful, and super easy-to-use **Personal Expense Tracker** built with Python and Flask. Its main goal is to help you take back control of your finances without dealing with cluttered spreadsheets or overly complex budgeting apps. 

Just log what you spend, see where your cash goes, and stay within your budget.

## ✨ Why this tracker?
- **It’s dead simple:** Add an expense in 5 seconds. Nothing more, nothing less. 
- **Keeps you in check:** Set a monthly budget (like 50,000 units), and the dashboard will politely (but firmly!) alert you if you're spending way too much. 🚨
- **Visualizes your habits:** Beautiful, interactive charts show exactly which categories—like Food, Travel, or Bills—are eating up your cash.
- **Fast & Searchable:** Trying to remember what you paid for the internet last month? The built-in live search bar finds it instantly.

## 🚀 How to run it locally

Want to give it a spin yourself? It's super easy to get everything running on your own computer.

### Prerequisites
Make sure you have [Python](https://www.python.org/) installed!

### Setup Instructions
1. **Clone this repository** to your local machine.
2. **Open your terminal** and navigate into the folder:
   ```bash
   cd expense-tracker
   ```
3. **Install the required dependencies** (we kept it lightweight!):
   ```bash
   pip install -r requirements.txt
   ```
4. **Fire it up!**
   ```bash
   python app.py
   ```
5. **Open Microsoft Edge** (or your browser of choice) and go to `http://127.0.0.1:5000`. You're all set!

## 🛠️ Built With Love Using...
- **Backend:** Python + Flask
- **Database:** SQLite (powered by Flask-SQLAlchemy)
- **Frontend UI:** HTML5 + Bootstrap 5 (for those sleek, responsive cards and buttons)
- **Data Visualizations:** Chart.js (for the awesome pie and bar charts)
